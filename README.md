# Stockton & Rogers Holding Page

Premium holding page for Stockton & Rogers Functional Art.

## Live Site
https://coyotes0g.github.io/stockton-rogers-holding-page/

## How to customize

1. Replace the placeholder images in `index.html` with your actual product photos and logo.
2. Update the email and phone if needed.
3. For production, host the images on your own server or CDN and update the `src` attributes.
4. Deploy to GitHub Pages (already enabled) or your preferred host.
